Phase 0: Foundation & Core Web Skills

Establish solid front-end fundamentals before transitioning into frameworks and dynamic builds.

  1. Home
  2. Phase 0

Technical Objectives

  • Build semantic, standards-compliant HTML layouts for responsive design.
  • Apply modern CSS (Flexbox, Grid) for scalable page structures.
  • Write clean JavaScript for DOM interaction and component behaviors.
  • Use Git & GitHub to manage versions and document your workflow.

LinkedIn Learning Courses

Learn to craft semantic, accessible markup and structure web pages that serve as clean foundations for future React components.

Master responsive design, layout grids, and visual hierarchy — foundational for consistent playbook UI components.

Explore variables, loops, and event handling to bring interactivity to static pages — the skills React will later abstract.

Learn professional version-control practices — branching, merging, and commit discipline that mirror real dev-team workflows.

Strengthen problem-solving skills through efficient search and documentation lookup — essential for debugging and learning APIs.

Milestone Projects

  • Project 1: Rebuild the Healthcare Playbook homepage using pure HTML5 and CSS3 (no Bootstrap).
  • Project 2: Add interactivity with vanilla JavaScript (expandable persona sections, modal details).
  • Project 3: Commit iterative versions to GitHub, tagging major checkpoints (v0.1, v0.2, v1.0).

KMBS Application Context

Phase 0 directly supports your work on Konica Minolta’s Sales Enablement Playbooks by building a foundation in front-end craftsmanship. As you learn semantic markup, layout systems, and Git discipline, you’ll also begin documenting and modularizing the existing playbook components that will later evolve into React UI modules.

Phase Duration

≈ 3 weeks (part-time)

Primary Languages

HTML · CSS · JavaScript · Git
