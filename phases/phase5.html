Mini Code Snippet

// Example: Auto-link your repo stars to your portfolio
fetch("https://api.github.com/users/BradleyAPierce/repos")
  .then(response => response.json())
  .then(data => {
    const repoList = document.getElementById("repo-list");
    data.forEach(repo => {
      if (!repo.fork) {
        const item = document.createElement("li");
        item.innerHTML = `⭐ <a href="${repo.html_url}" target="_blank">${repo.name}</a> – ${repo.stargazers_count} stars`;
        repoList.appendChild(item);
      }
    });
  });

This lightweight script dynamically fetches your GitHub repositories and displays them with live star counts — a simple but elegant way to showcase your active development footprint directly inside your portfolio.
